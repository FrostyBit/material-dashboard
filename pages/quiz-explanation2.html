Quiz Explanation

1. Which of the following are examples of end devices?

Answer: Workstations and Web Server
Explanation: End devices are devices that a person uses for communication, like workstations and web servers. Networking devices like routers or switches are not considered end devices.

2. Which of the following is a valid IPv4 address?

Answer: 255.255.255.1
Explanation: A valid IPv4 address is in the format of four numbers separated by periods, with each number ranging from 0 to 255. `255.255.255.1` is a valid address, whereas `1.12.0.256` is invalid due to the number `256`.

3. Which port number is associated with HTTP?

Answer: 80
Explanation: Port numbers are used to identify specific services or applications running on a device within a network. These numbers help direct traffic to the correct service. The commonly known port numbers and their associated services include:
  • Port 80 is the default port for HTTP (HyperText Transfer Protocol). HTTP is the protocol used for transferring web pages and resources over the internet in an unencrypted format. When you visit a website without encryption (HTTP://), the browser communicates with the server over port 80.
  • Port 22 is associated with SSH (Secure Shell), used for secure remote access to servers.
  • Port 53 is used for DNS (Domain Name System) queries, which resolve domain names into IP addresses.
  • Port 110 is used by POP3 (Post Office Protocol version 3) for receiving email.
Each service runs on a specific port, and knowing these port numbers helps with network troubleshooting and configuration. HTTP's use of port 80 is standardized across the internet.

4. What is the purpose of a subnet mask?

Answer: To determine the network portion of an IP address
Explanation: A subnet mask is used to determine the network portion of an IP address. It helps devices on the same network communicate with each other and differentiate between the network and host portions of an IP address.

5. What does a MAC address represent?

Answer: Physical address of a device
Explanation: A MAC address is a unique identifier assigned to network interfaces for communications on the physical network segment. It is used to identify devices on a network.
